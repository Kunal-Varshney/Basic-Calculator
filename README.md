# 🧮 Calculator App

A simple and user-friendly **Calculator App** built with **HTML, CSS, and JavaScript**.
It allows users to perform basic arithmetic operations with a clean and responsive interface.

---

## 🚀 Features

* Perform basic calculations (Addition, Subtraction, Multiplication, Division)
* Real-time input display
* Clear and delete functionality
* Responsive and clean UI design
* Smooth button interactions

---

## 🛠️ Tech Stack

* **HTML** → Structure of the app
* **CSS** → Styling and layout
* **JavaScript** → Logic and calculations

---

## 🎯 How to Use

1. Open the calculator in your browser
2. Click on numbers and operators to input values
3. Press **=** to get the result
4. Use **C / AC** to clear the screen
5. Continue calculations as needed
